## ✨ Overviewkk

**Imagyn** is a **personal digital gallery** for showcasing photos, certificates, hackathon snapshots, and nature clicks — built with the goal of evolving into a full-fledged **social photo-sharing platform** like Instagram.

The app is currently live at:
👉 **[imagyn-swart.vercel.app](https://imagyn-swart.vercel.app/)**

It features a dynamic hero section, categorized photo collections, lightbox view, and detail modals — all crafted for a smooth, professional UX.

| Category | Technologies |
|-----------|---------------|
| **Frontend** | [React](https://react.dev/), [Vite](https://vitejs.dev/), [Bootstrap 5](https://getbootstrap.com/) |

| **Deployment** | [Vercel](https://vercel.com/) |
| **Package Manager** | npm |
| **State & Data** | React Hooks, localStorage (for image details) |
| **Language** | JavaScript (ES6+) |

## 🎨 Features

✅ **Hero Section** with animation and dynamic welcome text  
✅ **Three Gallery Sections:**
   - Certificates 📜  
   - Coding Area 💻  
   - Miscellaneous 🌿  

✅ **Interactive Image Grid** with hover and zoom effects  
✅ **Lightbox View** — click image to view in full screen  
✅ **Details Modal** — read-only description for each image  
✅ **Responsive UI** for mobile, tablet, and desktop  
✅ **LocalStorage Support** for saving image details (editable via admin component)  
✅ **Deployed on Vercel** with blazing fast global CDN delivery
